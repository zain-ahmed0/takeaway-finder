# Takeaway Finder

A web application that helps users find nearby takeaway restaurants using Google Maps integration and the Just Eat Takeaway API.

## Getting Started

1. Create a Maps API Key on Google Cloud Platform
2. Restrict the Maps API Key to Maps JavaScript API
3. Enable Maps JavaScript API
4. Create a vector Map ID
5. Enter the API key in `FindTakeaway.razor`
6. Enter the Map ID in `googleMapsInterop`
